# PrismCode Example Projects

A collection of example projects demonstrating PrismCode capabilities.

## Examples

### 1. Todo App

**Location:** `examples/todo-app/`

Basic todo application generated by PrismCode agents.

- React frontend
- Node.js backend
- Full CRUD operations

### 2. Blog Platform

**Location:** `examples/blog-platform/`

Multi-author blogging platform.

- User authentication
- Markdown support
- Comment system

### 3. E-commerce Store

**Location:** `examples/ecommerce/`

Simple e-commerce storefront.

- Product catalog
- Shopping cart
- Checkout flow

### 4. Chat Application

**Location:** `examples/chat-app/`

Real-time chat application.

- WebSocket messaging
- User rooms
- Message history

### 5. Dashboard Analytics

**Location:** `examples/dashboard/`

Analytics dashboard with charts.

- Real-time metrics
- Data visualization
- Export functionality

### 6. REST API Service

**Location:** `examples/rest-api/`

RESTful API service boilerplate.

- Express.js setup
- Swagger documentation
- JWT authentication

### 7. CLI Tool

**Location:** `examples/cli-tool/`

Command-line interface tool.

- Commander.js framework
- Interactive prompts
- Config file support

### 8. GitHub Bot

**Location:** `examples/github-bot/`

GitHub automation bot.

- Issue management
- PR automation
- Webhook handling

### 9. Microservices Demo

**Location:** `examples/microservices/`

Microservices architecture example.

- Service discovery
- API gateway
- Message queue

### 10. Full-Stack Starter

**Location:** `examples/fullstack-starter/`

Full-stack application template.

- Next.js frontend
- Prisma ORM
- PostgreSQL database

## Running Examples

```bash
# Clone the repository
git clone https://github.com/pkurri/prismcode.git

# Navigate to example
cd prismcode/examples/todo-app

# Install dependencies
npm install

# Run development server
npm run dev
```
